About this site

This is a minimal, modern static website served by an Express + TypeScript backend. It demonstrates how to organize public assets and keep backend logic separate from UI.

  • Express serves files from /public
  • Strong defaults: Helmet, Compression, CORS and logging
  • Zero build for the frontend — just HTML, CSS and JS
