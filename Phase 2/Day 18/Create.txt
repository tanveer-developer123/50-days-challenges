# SQLite3 - Create (C in CRUD) Cheat Sheet

SQLite3 me CREATE ka matlab hota hai naye tables aur unke structure design karna.  
Yani hum define karte hain ki kaunsi columns hongi, unka data type kya hoga, aur constraints (rules) kya lagane hain.

---

## 1. Simple table banana
```sql
CREATE TABLE users (
    id INTEGER,
    name TEXT,
    age INTEGER
);
````

---

## 2. Primary Key ke sath

```sql
CREATE TABLE users (
    id INTEGER PRIMARY KEY,
    name TEXT,
    age INTEGER
);
```

---

## 3. AUTOINCREMENT use karke

```sql
CREATE TABLE users (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name TEXT,
    age INTEGER
);
```

---

## 4. UNIQUE constraint

```sql
CREATE TABLE users (
    id INTEGER PRIMARY KEY,
    email TEXT UNIQUE
);
```

---

## 5. NOT NULL constraint

```sql
CREATE TABLE users (
    id INTEGER PRIMARY KEY,
    name TEXT NOT NULL
);
```

---

## 6. DEFAULT value set karna

```sql
CREATE TABLE users (
    id INTEGER PRIMARY KEY,
    status TEXT DEFAULT 'active'
);
```

---

## 7. Foreign Key use karna

```sql
CREATE TABLE orders (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id INTEGER,
    product TEXT,
    FOREIGN KEY(user_id) REFERENCES users(id)
);
```

---

## 8. IF NOT EXISTS ke sath

```sql
CREATE TABLE IF NOT EXISTS users (
    id INTEGER PRIMARY KEY,
    name TEXT
);
```

---

## 9. Temporary table banana

```sql
CREATE TEMPORARY TABLE temp_users (
    id INTEGER,
    name TEXT
);
```

---

## 10. Dusre table ke structure se banana

```sql
CREATE TABLE new_users AS SELECT * FROM users;
```
